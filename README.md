Docker Wordpress Add Meta to Head Demo
=======================================

使用`add_action('wp_head', ...)`可以把自定义的内容（比如meta）输出到`<head>...</head>`中的某处。

修改的文件： 
- WordPress-5.3.2/test.php
- WordPress-5.3.2/wp-includes/functions.php


```
npm run up
```
